Summary

整理Github上相關project

wp-pivot-indicator

原開發者做了pivot control, 與他溝通後請他放上github, 改成iOS page control的形式並且允許使用者設定點的顏色與大小

NSLogger-CocoaLumberjack-connector

結合了 CocoaLumberjack and NSLogger做出一個singleton的log system, 達到可以動態的enable/disable log message, log level, and log function(ex: file logger). 此project也有follow cocoapods 格式上傳以方便大家使用

ibeacon-swift-tutorial & HiBeacons

簡單的實作iBeacon通訊, 一邊模擬商品, 一邊模擬使用者手機app, 當使用者接近商品的時候會收到push “buy one sned one iPhone6 !!” 細節可以看這裡

iOSVideoHardwareEncodeFFMPEGDecode

實作iOS video hardware encode, 並且以一個preview 畫面(類似facetime)模擬client端, demo 如何用ffmpeg decode h264 raw data.

novocaine

利用audio unit 實驗webRTC echo cancellation的效果.
